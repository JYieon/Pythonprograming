머신러닝?
: 함수 구현(몸체)을 기계가 스스로 구현하도록 하는 것
 데이터(X, Y) 

 머신러닝을 한다 => 데이터를 분석, 시각화, 전처리 과정 필수
 양질의 데이터가 성능에 많은 영향을 미침.


아까 만든 테스트 데이터는 마지막 테스트때 사용해야 하는 데이터.
인공지능 모델을 훈련시키고 중간 성능 테스트시에는 테스트 데이터 사용하면 안됨

훈련데이터 훈련 
-> 테스트 데이터로 성능 측정 
-> 어? 성능이 안좋네? 이렇게 고쳐봐야지
-> 다시 훈련 데이터로  훈련
-> 테스트 데이터로 성능 측정 
=> 공정하게 개발되었다고 할 수 없음


훈련데이터 훈련 
-> 검증 데이터로 성능 측정 
-> 어? 성능이 안좋네? 이렇게 고쳐봐야지
-> 다시 훈련 데이터로  훈련
-> 검증 데이터로 성능 측정 
-> 아 이제 완벽한듯? 배포하자
-> 테스트 데이터로 평가

훈련데이터 : 문제집
검증데이터 : 모의고사
테스트데이터 : 수능
